# Quests To-Do

App de Tarefas feito para o Ensino de React na Ctrl+Play, aplicado no Ctrl+Young 3.

## Como executar

### Iniciar o projeto 

- `npm start`

Abre [http://localhost:3000](http://localhost:3000) no seu navegador

### Executar testes

- `npm test`

Executa os testes criados no `QuestItem.test.js` e `QuestList.test.js`.
